# Datasets

## Overview

This document provides detailed information on the various datasets utilized in the Thinking Dataset Project. These datasets are crucial for generating accurate and complex chains of thought reasoning for real-world Situational Reports (SitReps) and creating useful case studies based on socio-economic conditions and stakeholders.

## WikiLeaks Cablegate

**Description**: A collection of U.S. State Department diplomatic cables offering insights into diplomatic affairs, socio-economic conditions, and political events.

**Key Components**:
- Diplomatic communications
- Socio-economic conditions
- Political events

**Strengths**:
- Detailed insights into diplomatic affairs
- Historical context

**Access and Download**:
1. Visit the [WikiLeaks Cablegate](https://wikileaks.org/cablegate.html) website.
2. Search for the "Cablegate dataset."
3. Download the ZIP file containing the cables.
4. Extract the files and store them in a structured format (e.g., CSV).

**Estimated Size**: 2,325,961 files

## GDELT

**Description**: The Global Database of Events, Language, and Tone; a resource for real-time global events, news coverage, and societal trends.

**Key Components**:
- Real-time global events
- News coverage
- Societal trends

**Strengths**:
- Extensive coverage
- Real-time updates

**Access and Download**:
1. Visit the [GDELT](https://www.gdeltproject.org/) website.
2. Navigate to the "Download" section.
3. Select and download the datasets.
4. Store the files in a structured format.

**Estimated Size**: 630,131,403 records

## ICB Project

**Key Components**:

**Strengths**:

**Access and Download**:

2. Navigate to the "Data" section.
3. Download the datasets.

- Targets

**Access and Download**:

**Estimated Size**: 181,691 entries
